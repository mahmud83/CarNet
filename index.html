Quick start

  1. For Web Server Installation & Running, refer to the WebServer-Installation link.
  2. Head to Home Page.
  3. To do a search on the Available Database to get live information at the current time, go to http://127.0.0.1:8000/rest/getInfo/[signalId]/.

For Complete Documentation, refer to Repository Wiki.
